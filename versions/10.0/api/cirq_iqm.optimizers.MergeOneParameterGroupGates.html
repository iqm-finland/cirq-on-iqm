You're reading an old version of this documentation. If you want up-to-date information, please have a look at 11.8.

cirq_iqm.optimizers.MergeOneParameterGroupGates#

class cirq_iqm.optimizers.MergeOneParameterGroupGates(post_clean_up=<function PointOptimizer.<lambda>>)#

Bases: cirq.circuits.optimization_pass.PointOptimizer

Merges adjacent gates belonging to the same parametrized gate family.

The merged gates have to act on the same sequence of qubits. This optimizer only works with gate families that are known to be one-parameter groups.

For now, all the families are assumed to be periodic with a period of 4.

Inits PointOptimizer.

Parameters
