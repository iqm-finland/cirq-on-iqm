cirq_iqm.iqm_sampler.IQMSampler¶

class cirq_iqm.iqm_sampler.IQMSampler(url, settings, device, qubit_mapping=None, **user_auth_args)¶

Bases: cirq.work.sampler.Sampler

Circuit sampler for executing quantum circuits on IQM quantum computers.

Parameters

  • url (str) – Endpoint for accessing the server interface. Has to start with http or https.

  • settings (str) – Settings for the quantum computer

  • device (IQMDevice) – Quantum architecture to execute the circuit on

  • qubit_mapping (dict[str, str]) – Injective dictionary that maps logical qubit names to physical qubit names

Keyword Arguments

  • auth_server_url – URL of user authentication server, if required by the IQM Cortex server. This can also be set in the IQM_AUTH_SERVER environment variable.

  • username – Username, if required by the IQM Cortex server. This can also be set in the IQM_AUTH_USERNAME environment variable.

  • password – Password, if required by the IQM Cortex server. This can also be set in the IQM_AUTH_PASSWORD environment variable.
