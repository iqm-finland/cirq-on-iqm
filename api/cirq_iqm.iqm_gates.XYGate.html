cirq_iqm.iqm_gates.XYGate¶

cirq_iqm.iqm_gates.XYGate(p)¶

Rotation around the XX+YY axis of the two-qubit Hilbert space.

Generated by the Hamiltonian \(H_{\text{XY}} = X \otimes X +Y \otimes Y\).

\[\text{XY}(p) = \exp(-i H_{\text{XY}} \: \frac{\pi}{2} \: p), \quad \text{where} \: p \in [0, 2).\]
Parameters

p (Union[float, sympy.core.basic.Basic]) – group parameter

Returns

corresponding Cirq gate
