submit_circuit(circuit, qubit_mapping, shots=1)¶

Submits a quantum circuit to be executed on a quantum computer.

Parameters

  • circuit (cirq_iqm.iqm_client.CircuitDTO) – circuit to be executed

  • qubit_mapping (list) – mapping of human-readable qubit names to physical qubit names

  • shots (int) – number of times the circuit is sampled

Returns

ID for the created task. This ID is needed to query the status and the execution results.

Return type

uuid.UUID
